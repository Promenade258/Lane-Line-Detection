# Lane-Line-Detection

### 代码描述
这个Python文件基于OpenCV和Matplotlib库实现**车道线检测**核心流程，主要针对单张道路图像完成从预处理到直线检测的全流程操作，具体功能如下：

1. **图像基础处理**：读取图像并转换为灰度图，为后续滤波和边缘检测做准备；
2. **多种滤波操作验证**：依次实现均值滤波、方框滤波（归一化/非归一化）、高斯滤波、中值滤波、双边滤波，对比不同核尺寸/参数下的滤波效果，并将滤波结果保存；
3. **边缘检测**：基于滤波后的图像，分别通过Sobel算子（X/Y梯度合并）和Canny算法提取图像边缘；
4. **感兴趣区域（ROI）提取**：通过梯形掩码筛选出图像中车道线所在的核心区域，剔除无关背景干扰；
5. **霍夫变换直线检测**：分别使用霍夫线变换（HoughLines）和概率霍夫线变换（HoughLinesP）检测ROI区域内的直线，并将检测到的车道线绘制在原始图像上；
6. **模块化封装**：将核心流程封装为`Complete_draw_lines`类，整合“读取图像→灰度转换→滤波+边缘检测→ROI提取→直线检测→结果绘制/保存”全流程，支持单张图像的自动化处理与结果可视化、保存。

### 核心应用场景
代码聚焦于**道路场景下的车道线检测**，可作为自动驾驶、辅助驾驶系统中车道线识别模块的基础实现，通过对比不同滤波/边缘检测/直线检测算法的效果，验证最优参数组合。

### 关键特性
- 可视化各步骤处理结果（滤波、边缘、ROI、检测结果）；
- 支持多种滤波/检测算法的参数对比实验；
- 模块化设计，便于扩展和参数调整；
- 自动保存各步骤处理结果及最终车道线检测图像。
